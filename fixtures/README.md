# Fixtures

`doc-source-map/pug-html` contains the first generated-source PugDoc fixture:

- `src/pages/card.pug` is the entry file.
- `src/layouts/base.pug` provides template inheritance.
- `src/partials/action.pug` provides an included mixin.
- `dist/card.html` is generated by Pug.
- `dist/card.pugdoc.json` is the PugDoc extraction artifact.
- `dist/card.htmdoc.json` is the HTMDoc bridge artifact.
- `dist/card.docmap.json` is the documentation source-map manifest.
- `dist/card.hia.json` is the HIA core document fixture.

Generate and check artifacts with:

```sh
npm run build:fixtures
npm run check:fixtures
```
